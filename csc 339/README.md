# Student Login & Information System

A web-based student information collection system with login validation functionality.

## Project Overview

This project consists of two main pages:
1. **Login Page** - Validates user credentials before granting access
2. **Student Information Page** - Collects and displays student data

Built as part of my web development coursework to practice form handling, validation, and page navigation.
